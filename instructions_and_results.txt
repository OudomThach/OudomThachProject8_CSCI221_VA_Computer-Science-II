Assignment Purpose
------------------
This project completes the instructor's requirement to create a template-based
class that implements a set of items. A set is a collection where no item occurs
more than once.

The Set class can:
1. Add a new item to the set.
2. Remove an item from the set.
3. Return the number of items in the set.
4. Determine if an item is a member of the set.
5. Return a pointer to a dynamically created array containing each item in the set.

Files Included
--------------
1. Set.h
   - Contains the template-based Set class.
   - Uses a dynamic array to store unique items.
   - Includes comments explaining the code.

2. main.cpp
   - Tests every required Set function.
   - Shows that duplicate items are rejected.
   - Shows that the template works with integers and strings.
   - Deletes the dynamic array returned by toArray().

3. instructions_and_results.txt
   - Gives compile instructions, run instructions, and test results.
