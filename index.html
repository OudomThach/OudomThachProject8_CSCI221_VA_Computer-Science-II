Assignment instruction

What the book asks for

Write a template-based class that implements a set of items. A set is a collection where no item occurs more than once. Internally, the set may use an array, vector, list, or another structure, but the class must externally support the required operations.

a

Add a new item

If the item is already in the set, nothing happens.

b

Remove an item

Remove the matching value when it exists in the set.

c

Return the count

getCurrentSize() returns the number of stored items.

d

Check membership

contains() determines whether an item is in the set.

e

Return an array pointer

toArray() returns a new dynamic array. The caller deallocates it.

test

Use different data types

The C++ test program creates Set<int> and Set<std::string>.
